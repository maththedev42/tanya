# 01 - Hello world

Use the smallest Tanya path first: one prompt, no tools.

## Prerequisites

- Node.js 20 or newer
- Tanya installed or linked locally
- `DEEPSEEK_API_KEY` set

## Run

```bash
tanya ask "hello"
```

## What to expect

Tanya sends one provider request and prints the answer directly. For a replayable
terminal transcript, see [`demo.cast`](./demo.cast).
