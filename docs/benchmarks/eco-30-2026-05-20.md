## Honest weak spots

`eco-19-long-file-read-dedup` timed out at the 10 minute per-task cap. It is the only non-pass in this run. No task errored before starting, so the startup/config error rate was 0/30.

The slowest passing tasks were also token-economy stress cases: `eco-25-long-file-read-dedup` (222.5s) and `eco-21-reasoning-heavy-planning` (193.1s).

## Methodology

- Suite: eco-30 (`src/eval/suites/`)
- Tanya version: 0.17.0-beta.2
- Per-task wall-clock cap: 10 min; token cap: 500k
- Sequential, single concurrency, real DeepSeek API
- Cost estimates use the run-log estimator formula with DeepSeek-V4-Pro cache-miss rates from DeepSeek pricing: $0.435 / 1M input tokens and $0.87 / 1M output tokens
- Run date: 2026-05-20
